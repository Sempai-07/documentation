# Interface: ISearchResult

Represents a search result for a specific message entity type.

## Properties

### index

> **index**: `number`

The index of the entity.

***

### length

> **length**: `number`

The length of the entity.

***

### offset

> **offset**: `number`

The starting offset of the entity.

***

### search

> **search**: `string`

The text that matches the entity.
