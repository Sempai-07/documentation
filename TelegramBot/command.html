Telegram Bot API Command Handler

Description:

Use this command handler to register custom command functions for your Telegram bot. The command handler listens for specific commands and executes the provided callback function when the command is received.

Parameters:

Parameter Description
command The command string or an array of command strings to listen for.
Type: string | string[]
callback The callback function to handle the command when received.
Type: (message: Message.TextMessage & Context<F>, args?: string[]) => void
typeChannel(optional) Optional. The type of channel to listen for commands in (e.g., "private", "group", "supergroup", "channel"). If not specified or set to false, the command will be listened to in all types of channels.
Type: "private" | "group" | "supergroup" | "channel" | false

Example:


const { TelegramBot } = require("telegramsjs");

const bot = new TelegramBot("BOT_TOKEN");

bot.command("start", (ctx) => {
  ctx.reply("Starts command, yaka is available in private messages!");
}, "private");

bot.command(["username", "chatid"], (ctx, args) => {
  if (args[0] === "/username") {
    const username = ctx.from.username ? `@${ctx.from.username}` : ctx.from.first_name;
    ctx.reply("Your username: " + username);
    return;
  }
  if (args[0] === "/chatid") {
    ctx.reply("Supergroup id: " + ctx.chat.id);
    return;
  }
}, "supergroup");

bot.login();
