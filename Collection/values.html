Collection<K, V>

Description:

This class represents a collection with key-value pairs. It uses an underlying map to store the data.

Method:

values(): IterableIterator<V>
Returns an iterator that contains all the values of the elements in the collection. 

        
const valuesIterator = collection.values();
for (const value of valuesIterator) {
  // Your code here to handle each value
}

Return Value:

An iterator that contains all the values of the elements in the collection.
