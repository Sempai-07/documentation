Collection<K, V>

Description:

This class represents a collection with key-value pairs. It uses an underlying map to store the data.

Method:

flatMap( callbackFn: (value: V, key: K, collection: Collection<K, V>) => U[], thisArg?: unknown, )
Applies a mapping function to each value in the collection and flattens the result. 

        
const flatMappedCollection = collection.flatMap((value, key, collection) => {
  // Your mapping function logic here to return an array of mapped values
  return [mappedValue1, mappedValue2, ...];
}, thisArg);

Parameters:

  • callbackFn: The mapping function to apply. It should take three arguments: the value, the key, and the collection instance. The function should return an array of mapped values.
  • thisArg: An optional value to use as this when calling the function.

Return Value:

A new Collection instance with the flattened mapped values.
