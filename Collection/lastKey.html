Collection<K, V>

Description:

This class represents a collection with key-value pairs. It uses an underlying map to store the data.

Method:

lastKey(): K | undefined
Retrieve the last key in the collection. 

        
const lastKey = collection.lastKey();
if (lastKey !== undefined) {
  // Your code here to handle the last key
} else {
  // Your code here to handle the case when the collection is empty
}

Return Value:

The last key in the collection, or undefined if the collection is empty.
