Collection<K, V>

Description:

This class represents a collection with key-value pairs. It uses an underlying map to store the data.

Method:

toMap(): Map<K, V>
Returns an instance of the Map class that contains all the elements of the given collection.

// Your code here to use the map instance

Return Value:

An instance of the Map class containing all the elements of the given collection.
