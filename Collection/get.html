Collection<K, V>

Description:

This class represents a collection with key-value pairs. It uses an underlying map to store the data.

Constructor:

        
const collection = new Collection([
  [key1, value1],
  [key2, value2],
  // Add more key-value pairs as needed
]);

Properties:

  • size: The number of key-value pairs in the collection.

Methods:

get(key: K): V | undefined
Retrieve the value associated with the given key. Returns the value associated with the key or undefined if the key is not in the collection. 

        
const value = collection.get(key);
console.log(value); // Output: The value associated with the key or undefined
