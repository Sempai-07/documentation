Collection<K, V>

Description:

This class represents a collection with key-value pairs. It uses an underlying map to store the data.

Method:

last(): V | undefined
Retrieve the last value in the collection. 

        
const lastValue = collection.last();
if (lastValue !== undefined) {
  // Your code here to handle the last value
} else {
  // Your code here to handle the case when the collection is empty
}

Return Value:

The last value in the collection, or undefined if the collection is empty.
