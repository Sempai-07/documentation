Collection<K, V>

Description:

This class represents a collection with key-value pairs. It uses an underlying map to store the data.

Method:

map<U>( callbackFn: (value: V, key: K, collection: Collection<K, V>) => U, thisArg?: unknown, )
Creates a new collection with the results of calling a provided function on every element in the calling collection. 

        
const mappedCollection = collection.map((value, key, collection) => {
  // Your function logic here, return the new value
  return newValue;
});

Parameters:

  • callbackFn: Function that produces an element of the new collection. It should take three arguments: the value, the key, and the collection instance.
  • thisArg: An optional object to use as this when executing the callback.

Return Value:

A new collection with the results of calling the callback on each element.
