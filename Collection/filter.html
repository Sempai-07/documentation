Collection<K, V>

Description:

This class represents a collection with key-value pairs. It uses an underlying map to store the data.

Method:

filter( callbackFn: (value: V, key: K, collection: Collection<K, V>) => boolean, thisArg?: unknown, )
Create a new collection that includes only the key-value pairs that satisfy a given condition. The condition should take three arguments: the value, the key, and the collection instance. Return true to include the pair, false to exclude it. 

        
const filteredCollection = collection.filter((value, key, collection) => {
  // Your condition logic here
  return true; // include the pair, or false to exclude it
});

Parameters:

  • callbackFn: The condition to test each pair against. It should take three arguments: the value, the key, and the collection instance. Return true to include the pair, false to exclude it.
  • thisArg: An optional value to use as this when calling the function.

Return Value:

A new Collection instance containing the filtered key-value pairs.
