Collection<K, V>

Description:

This class represents a collection with key-value pairs. It uses an underlying map to store the data.

Method:

tap(callbackFn: (value: V, key: K, collection: Collection<K, V>) => void): Collection<K, V>
Perform an operation on each value in the collection and return the collection. 

        
collection.tap((value, key, collection) => {
  // Your code here to perform an operation on each value
});
// Your code here to continue with the collection

Parameters:

  • callbackFn: The function to perform on each value. It should take three arguments: the value, the key, and the collection instance.

Return Value:

The collection instance (for chaining).
