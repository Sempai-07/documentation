Collection<K, V>

Description:

This class represents a collection with key-value pairs. It uses an underlying map to store the data.

Method:

partition(callbackFn: Function, thisArg?: unknown): GroupedValues<K, V>[]
Create a new collection by partitioning the elements into two groups based on a condition. 

        
const partitionedGroups = collection.partition((value, key, collection) => {
  // Your condition to test each element and return true/false
}, thisArg);
// Your code here to handle the partitioned groups

Parameters:

  • callbackFn: The condition to test each element against. It should take three arguments: the value, the key, and the collection instance. Return true to include the element in the first group, false to include it in the second group.
  • thisArg: An optional value to use as `this` when calling the function.

Return Value:

An array of two `GroupedValues` objects representing the partitioned groups.
