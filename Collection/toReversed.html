Collection<K, V>

Description:

This class represents a collection with key-value pairs. It uses an underlying map to store the data.

Method:

toReversed(): Collection<K, V>
Returns a new Collection instance with the key-value pairs in reversed order. 

        
const reversedCollection = collection.toReversed();
// Your code here to work with the reversedCollection

Return Value:

A new Collection instance with the reversed key-value pairs.
