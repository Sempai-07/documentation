Collection<K, V>

Description:

This class represents a collection with key-value pairs. It uses an underlying map to store the data.

Method:

keys(): IterableIterator<K>
Returns an iterator that contains all the keys of the collection's elements. 

        
const keysIterator = collection.keys();
for (const key of keysIterator) {
  // Your code here to handle each key
}

Return Value:

An iterator that contains all the keys of the collection's elements.
