Collection<K, V>

Description:

This class represents a collection with key-value pairs. It uses an underlying map to store the data.

Method:

toArrayByKey(): GroupedValues<K, V>[]
Returns an array that contains all the values of the elements of the collection, grouped by keys. 

        
const groupedValuesArray = collection.toArrayByKey();
// Your code here to work with the array

Return Value:

An array that contains all the values of the elements of the collection, grouped by keys.
