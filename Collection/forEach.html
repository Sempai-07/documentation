Collection<K, V>

Description:

This class represents a collection with key-value pairs. It uses an underlying map to store the data.

Method:

forEach( callbackFn: (value: V, key: K, collection: Collection<K, V>) => void, thisArg?: unknown, )
Call a given function for each key-value pair in the collection. The function should take three arguments: the value, the key, and the collection instance. 

        
collection.forEach((value, key, collection) => {
  // Your custom logic here
});

Parameters:

  • callbackFn: The function to call for each pair. It should take three arguments: the value, the key, and the collection instance.
  • thisArg: An optional value to use as this when calling the function.

Return Value:

This method does not have a return value (void).
