Collection<K, V>

Description:

This class represents a collection with key-value pairs. It uses an underlying map to store the data.

Method:

toJSON(): { [key: string]: V }
Returns an object that contains all the elements of the collection as properties with keys in string format. 

        
const json = collection.toJSON();
// Your code here to work with the json object

Return Value:

An object containing the elements of the collection with keys in string format.
