Collection<K, V>

Description:

This class represents a collection with key-value pairs. It uses an underlying map to store the data.

Method:

subtract(collection: Collection<K, V>): Collection<K, V>
Subtracts the elements of the given collection from this collection. 

        
const otherCollection = new Collection();
// Add some entries to otherCollection

collection.subtract(otherCollection);
// Your code here to handle the modified collection (collection)

Parameters:

  • collection: The collection to subtract. It should be of type Collection<K, V>.

Return Value:

The modified Collection instance (for chaining).
