Collection<K, V>

Description:

This class represents a collection with key-value pairs. It uses an underlying map to store the data.

Method:

delete(key: K): boolean
Remove the key-value pair associated with the given key. Returns true if the key-value pair was removed, false if the key was not in the collection. 

        
const wasDeleted = collection.delete(key);
console.log(wasDeleted); // Output: true or false

Parameter:

  • key: The key to remove.

Return Value:

True if the key-value pair was removed, false if the key was not in the collection.
