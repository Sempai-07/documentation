Collection<K, V>

Description:

This class represents a collection with key-value pairs. It uses an underlying map to store the data.

Method:

reduce<U>( callbackFn: ( accumulator: U, value: V, key: K, collection: Collection<K, V>, ) => U, initialValue?: U, )
Applies a function against an accumulator and each element in the collection to reduce it to a single value. 

        
const reducedValue = collection.reduce((accumulator, value, key, collection) => {
  // Your function logic here to reduce the value
  return reducedValue;
}, initialValue);

Parameters:

  • callbackFn: Function to execute on each element in the collection. It should take four arguments: the accumulator, the value, the key, and the collection instance.
  • initialValue: Value to use as the first argument to the first call of the callback (optional).

Return Value:

The reduced value.
