Collection<K, V>

Description:

This class represents a collection with key-value pairs. It uses an underlying map to store the data.

Method:

chunk(size: number): [number, V][]
Splits the collection into subcollections of the given size and returns an array with new collections. 

        
const size = 3;
const chunks = collection.chunk(size);
// Your code here to handle the array with new collections

Parameters:

  • size: The amount by which this class will be divided. It should be of type number.

Return Value:

An array with new collections, each containing key-value pairs from the original collection, divided into subcollections of the given size.
