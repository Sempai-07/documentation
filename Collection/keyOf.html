Collection<K, V>

Description:

This class represents a collection with key-value pairs. It uses an underlying map to store the data.

Method:

keyOf(value: V): K | undefined
Returns the first key corresponding to the specified value in the collection. 

        
const valueToFind = ...; // specify the value to look for
const key = collection.keyOf(valueToFind);
// Your code here to handle the returned key

Parameters:

  • value: The value to look for in the collection. It should be of type V.

Return Value:

The first key corresponding to the specified value in the collection, or undefined if the value is not found.
