Collection<K, V>

Description:

This class represents a collection with key-value pairs. It uses an underlying map to store the data.

Method:

has(key: K): boolean
Check whether the collection contains a given key. Returns true if the collection contains the key, false otherwise. 

        
const hasKey = collection.has(key);
console.log(hasKey); // Output: true or false

Parameter:

  • key: The key to look for.

Return Value:

True if the collection contains the key, false otherwise.
