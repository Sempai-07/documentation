Collection<K, V>

Description:

This class represents a collection with key-value pairs. It uses an underlying map to store the data.

Method:

at(index: number): V | undefined
Retrieve the value at the specified index in the collection. 

        
const index = 2; // Replace with the desired index
const valueAtIndex = collection.at(index);
if (valueAtIndex !== undefined) {
  // Your code here to handle the value at the specified index
} else {
  // Your code here to handle the case when the index is out of range
}

Parameters:

  • index: The index of the value to retrieve. Use a positive index to access from the beginning, and a negative index to access from the end (e.g., -1 for the last element).

Return Value:

The value at the specified index, or undefined if the index is out of range.
