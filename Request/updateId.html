Telegram Bot API "updateId" Property

Description:

Use this property to get the last update ID received by the Telegram bot. The "updateId" property provides the ID of the last update processed by the bot. It returns the update ID as a number or null if not available.

Returns:

The last update ID received by the bot, or null if not available.

Example:


// Example of using the "updateId" property to get the last update ID

const { TelegramBot } = require("telegramsjs");
const bot = new TelegramBot("BOT_TOKEN");

// Example function to get the last update ID using the "updateId" property
function getLastUpdateId() {
  const lastUpdateId = bot.updateId;
  console.log(`Last Update ID: ${lastUpdateId}`);
}

// Call the function to get the last update ID
getLastUpdateId().then(data => console.log(data));

bot.login();
