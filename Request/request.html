Telegram Bot API "request" Function

Description:

Use this function to make a request to the Telegram Bot API. The "request" function performs the API call using the provided method and parameters. It returns a Promise that resolves with the response from the API call.

Parameters:

Parameter Description
method The API method to call.
Type: string
params Optional. The parameters to include in the API call.
Type: object

Returns:

A Promise that resolves with the response from the API call as a TelegramApiResponse object.

Example:


// Example of using the "request" function to make an API call

const { TelegramBot } = require("telegramsjs");
const bot = new TelegramBot("BOT_TOKEN");

// Example API call using the "request" function
async function getMe() {
  try {
    const response = await bot.request("getMe");
    console.log(response);
  } catch (error) {
    console.error(error);
  }
}

// Call the function to make the API call
