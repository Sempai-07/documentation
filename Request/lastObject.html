Telegram Bot API "lastObject" Property

Description:

Use this property to get the last object received by the Telegram bot. The "lastObject" property provides the last received object, which could be an update or any other type of message from the Telegram Bot API.

Returns:

The last received object as an Update or undefined if no object has been received yet.

Example:


// Example of using the "lastObject" property to get the last received object

const { TelegramBot } = require("telegramsjs");
const bot = new TelegramBot("BOT_TOKEN");

// Example function to get the last received object using the "lastObject" property
function getLastReceivedObject() {
  const lastReceivedObject = bot.lastObject;
  console.log(`Last Received Object:`, lastReceivedObject);
}

// Call the function to get the last received object
getLastReceivedObject().then(data => console.log(data));

bot.login();
