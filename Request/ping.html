Telegram Bot API "ping" Function

Description:

Use this function to get the ping latency of the Telegram bot. The "ping" function sends a request to the Telegram Bot API and calculates the time taken for the response. It returns a Promise that resolves with the ping latency in milliseconds.

Returns:

A Promise that resolves with the ping latency of the bot in milliseconds.

Example:


// Example of using the "ping" function to get the bot's ping latency

const { TelegramBot } = require("telegramsjs");
const bot = new TelegramBot("BOT_TOKEN");

// Example function to get the bot's ping latency using the "ping" function
async function getBotPing() {
  try {
    const pingLatency = await bot.ping();
    console.log(`Bot ping latency: ${pingLatency} milliseconds`);
  } catch (error) {
    console.error(error);
  }
}

// Call the function to get the bot's ping latency
getBotPing().then(data => console.log(data));
