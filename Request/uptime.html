Telegram Bot API "uptime" Property

Description:

Use this property to get the uptime of the Telegram bot. The "uptime" property calculates the time elapsed since the bot started, returning the uptime in milliseconds.

Returns:

The uptime of the bot in milliseconds.

Example:


// Example of using the "uptime" property to get the bot's uptime

const { TelegramBot } = require("telegramsjs");
const bot = new TelegramBot("BOT_TOKEN");

// Get the bot's uptime using the "uptime" property
const botUptime = bot.uptime;
console.log(`Bot uptime: ${botUptime} milliseconds`);

bot.login();
