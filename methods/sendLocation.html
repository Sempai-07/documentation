sendLocation - Telegram Bot API

Description:

Use this method to send a point on the map. On success, the sent Message is returned.

Parameter Description
chat_id Unique identifier for the target chat or username of the target channel (in the format @channelusername)
Type: number | string
message_thread_id(optional) Unique identifier for the target message thread (topic) of the forum; for forum supergroups only
Type: number
latitude Latitude of the location
Type: number
longitude Longitude of the location
Type: number
horizontal_accuracy(optional) The radius of uncertainty for the location, measured in meters; 0-1500
Type: number
live_period(optional) Period in seconds for which the location will be updated (see Live Locations, should be between 60 and 86400)
Type: number
heading(optional) The direction in which the user is moving, in degrees; 1-360. For active live locations only.
Type: number
proximity_alert_radius(optional) The maximum distance for proximity alerts about approaching another chat member, in meters. For sent live locations only.
Type: number
disable_notification(optional) Sends the message silently. Users will receive a notification with no sound.
Type: boolean
protect_content(optional) Protects the contents of the sent message from forwarding and saving
Type: boolean
reply_to_message_id(optional) If the message is a reply, ID of the original message
Type: number
allow_sending_without_reply(optional) Pass True if the message should be sent even if the specified replied-to message is not found
Type: boolean

Return Values:

A Message.Location object representing the sent location message.

Example:


const { TelegramBot } = require("telegramsjs");
const bot = new TelegramBot("BOT_TOKEN");

bot.sendLocation({
  chat_id: 123456789,
  latitude: 40.7128,
  longitude: -74.0060,
}).then((data) => {
  console.log(data);
});

bot.login();
