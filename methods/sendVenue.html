sendVenue - Telegram Bot API

Description:

Use this method to send information about a venue. On success, the sent Message is returned.

Parameter Description
chat_id Unique identifier for the target chat or username of the target channel (in the format @channelusername)
Type: number | string
message_thread_id(optional) Unique identifier for the target message thread (topic) of the forum; for forum supergroups only
Type: number
latitude Latitude of the venue
Type: number
longitude Longitude of the venue
Type: number
title Name of the venue
Type: string
address Address of the venue
Type: string
foursquare_id(optional) Foursquare identifier of the venue
Type: string
foursquare_type(optional) Foursquare type of the venue, if known. (For example, “arts_entertainment/default”, “arts_entertainment/aquarium” or “food/icecream”.)
Type: string
google_place_id(optional) Google Places identifier of the venue
Type: string
google_place_type(optional) Google Places type of the venue. (See supported types.)
Type: string
disable_notification(optional) Sends the message silently. Users will receive a notification with no sound.
Type: boolean
protect_content(optional) Protects the contents of the sent message from forwarding and saving
Type: boolean
reply_to_message_id(optional) If the message is a reply, ID of the original message
Type: number
allow_sending_without_reply(optional) Pass True if the message should be sent even if the specified replied-to message is not found
Type: boolean
reply_markup(optional) An object for an inline keyboard, custom reply keyboard, instructions to remove reply keyboard, or to force a reply from the user.
Type: InlineKeyboardMarkup | ReplyKeyboardMarkup | ReplyKeyboardRemove | ForceReply

Return Values:

The Message.VenueMessage object representing the sent venue message.

Example:


const { TelegramBot } = require("telegramsjs");
const bot = new TelegramBot("BOT_TOKEN");

bot.sendVenue({
  chat_id: 123456789,
  latitude: 40.7128,
  longitude: -74.0060,
  title: "Central Park",
  address: "New York, NY, USA",
}).then((data) => {
  console.log(data);
});
