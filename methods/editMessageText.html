editMessageText - Telegram Bot API

Description:

Use this method to edit text and game messages. On success, if the edited message is not an inline message, the edited Message is returned, otherwise True is returned.

Parameters:

Parameter Description
chat_id(optional) Required if inline_message_id is not specified. Unique identifier for the target chat or username of the target channel (in the format @channelusername).
Type: number | string
message_id(optional) Required if inline_message_id is not specified. Identifier of the message to edit.
Type: number
inline_message_id(optional) Required if chat_id and message_id are not specified. Identifier of the inline message.
Type: string
text New text of the message, 1-4096 characters after entities parsing.
Type: string
parse_mode(optional) Mode for parsing entities in the message text. See formatting options for more details.
Type: ParseMode
entities(optional) A list of special entities that appear in message text, which can be specified instead of parse_mode.
Type: MessageEntity[]
disable_web_page_preview(optional) Disables link previews for links in this message.
Type: boolean
reply_markup(optional) An object for an inline keyboard.
Type: InlineKeyboardMarkup

Return Value:

Returns (Update.Edited & Message.TextMessage) | true on success.

Example:


const { TelegramBot } = require("telegramsjs");
const bot = new TelegramBot("BOT_TOKEN");

// Edit a message text
const editedMessage = bot.editMessageText({
  chat_id: 123456789, // Replace with the chat ID
  message_id: 987654321, // Replace with the message ID
  text: "New edited message text",
  parse_mode: 'MarkdownV2'
});
console.log(editedMessage);
