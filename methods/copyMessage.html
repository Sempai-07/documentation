copyMessage - Telegram Bot API

Description:

Use this method to copy messages of any kind. Service messages and invoice messages can't be copied. A quiz poll can be copied only if the value of the field correct_option_id is known to the bot. The method is analogous to the method forwardMessage, but the copied message doesn't have a link to the original message. Returns the MessageId of the sent message on success.

Parameter Description
chat_id Unique identifier for the target chat or username of the target channel (in the format @channelusername)
Type: number | string
message_thread_id(optional) Unique identifier for the target message thread (topic) of the forum; for forum supergroups only
Type: number
from_chat_id Unique identifier for the chat where the original message was sent (or channel username in the format @channelusername)
Type: number | string
message_id Message identifier in the chat specified in from_chat_id
Type: number
caption(optional) New caption for media, 0-1024 characters after entities parsing. If not specified, the original caption is kept
Type: string
parse_mode(optional) Mode for parsing entities in the new caption. See formatting options for more details.
Type: string
caption_entities(optional) A list of special entities that appear in the new caption, which can be specified instead of parse_mode
Type: MessageEntity[]
disable_notification(optional) Sends the message silently. Users will receive a notification with no sound.
Type: boolean
protect_content(optional) Protects the contents of the sent message from forwarding and saving
Type: boolean
reply_to_message_id(optional) If the message is a reply, ID of the original message
Type: number
allow_sending_without_reply(optional) Pass True if the message should be sent even if the specified replied-to message is not found
Type: boolean
reply_markup(optional) Additional interface options. An object for an inline keyboard, custom reply keyboard, instructions to remove reply keyboard, or to force a reply from the user.
Type: InlineKeyboardMarkup | ReplyKeyboardMarkup | ReplyKeyboardRemove | ForceReply

Return Value:

MessageId 


const { TelegramBot } = require("telegramsjs");
        
const bot = new TelegramBot("BOT_TOKEN");

  message_id: 123456789,
  from_chat_id: 123456789,
  chat_id: 123456789
}).then((data) => {
  console.log(data);
