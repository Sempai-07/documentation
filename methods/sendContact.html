sendContact - Telegram Bot API

Description:

Use this method to send phone contacts. On success, the sent Message is returned.

Parameter Description
chat_id Unique identifier for the target chat or username of the target channel (in the format @channelusername)
Type: number | string
message_thread_id(optional) Unique identifier for the target message thread (topic) of the forum; for forum supergroups only
Type: number
phone_number Contact's phone number
Type: string
first_name Contact's first name
Type: string
last_name(optional) Contact's last name
Type: string
vcard(optional) Additional data about the contact in the form of a vCard, 0-2048 bytes
Type: string
disable_notification(optional) Sends the message silently. Users will receive a notification with no sound.
Type: boolean
protect_content(optional) Protects the contents of the sent message from forwarding and saving
Type: boolean
reply_to_message_id(optional) If the message is a reply, ID of the original message
Type: number
allow_sending_without_reply(optional) Pass True if the message should be sent even if the specified replied-to message is not found
Type: boolean
reply_markup(optional) An object for an inline keyboard, custom reply keyboard, instructions to remove keyboard, or to force a reply from the user.
Type: InlineKeyboardMarkup | ReplyKeyboardMarkup | ReplyKeyboardRemove | ForceReply

Return Values:

The Message.ContactMessage object representing the sent contact message.

Example:


const { TelegramBot } = require("telegramsjs");
const bot = new TelegramBot("BOT_TOKEN");

bot.sendContact({
  chat_id: 123456789,
  phone_number: "+1234567890",
  first_name: "John",
  last_name: "Doe",
}).then((data) => {
  console.log(data);
});
