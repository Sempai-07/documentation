createInvoiceLink - Telegram Bot API

Description:

Use this method to create a link for an invoice. Returns the created invoice link as a String on success.

Parameter Description
title Product name, 1-32 characters
Type: string
description Product description, 1-255 characters
Type: string
payload Bot-defined invoice payload, 1-128 bytes. This will not be displayed to the user, use for your internal processes.
Type: string
provider_token Payment provider token, obtained via BotFather
Type: string
currency Three-letter ISO 4217 currency code, see more on currencies
Type: string
prices Price breakdown, a list of components (e.g. product price, tax, discount, delivery cost, delivery tax, bonus, etc.)
Type: LabeledPrice
max_tip_amount(optional) The maximum accepted amount for tips in the smallest units of the currency (integer, not float/double). For example, for a maximum tip of US$ 1.45 pass max_tip_amount = 145. See the exp parameter in currencies.json, it shows the number of digits past the decimal point for each currency (2 for the majority of currencies). Defaults to 0
Type: number
suggested_tip_amounts(optional) An array of suggested amounts of tips in the smallest units of the currency (integer, not float/double). At most 4 suggested tip amounts can be specified. The suggested tip amounts must be positive, passed in a strictly increased order, and must not exceed max_tip_amount.
Type: number[]
provider_data(optional) Data about the invoice, which will be shared with the payment provider. A detailed description of required fields should be provided by the payment provider.
Type: string
photo_url(optional) URL of the product photo for the invoice. Can be a photo of the goods or a marketing image for a service.
Type: string
photo_size(optional) Photo size in bytes
Type: number
photo_width(optional) Photo width
Type: number
photo_height(optional) Photo height
Type: number
need_name(optional) Pass True if you require the user's full name to complete the order
Type: boolean
need_phone_number(optional) Pass True if you require the user's phone number to complete theorder
Type: boolean
need_email(optional) Pass True if you require the user's email address to complete theorder
Type: boolean
need_shipping_address(optional) Pass True if you require the user's shipping address to complete theorder
Type: boolean
send_phone_number_to_provider(optional) Pass True if the user's phone number should be sent to the provider
Type: boolean
send_email_to_provider(optional) Pass True if the user's email address should be sent to the provider
Type: boolean
is_flexible(optional) Pass True if the final price depends on the shipping method
Type: boolean

Return Type:

string

Usage:


const { TelegramBot } = require("telegramsjs");

const bot = new TelegramBot("BOT_TOKEN");

bot.createInvoiceLink({
  title: "Example Product",
  description: "This is an example product",
  payload: "example_payload",
  provider_token: "YOUR_PROVIDER_TOKEN",
  currency: "USD",
  prices: [{ label: "Product Price", amount: 1000 }]
}).then((link) => {
  console.log("Invoice Link:", link);
});
