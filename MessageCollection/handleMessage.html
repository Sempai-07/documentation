MessageCollector - Telegram Bot API

Description:

A utility class that collects messages from a specific chat based on the provided criteria. It handles new messages received by the collector and emits a "collected" event with the collected messages and count.

Constructor Parameters:

Parameter Description
options An object containing options for the MessageCollector.
Type: object
options.chatId The ID of the chat you want to collect messages from.
Type: number
options.filter A filter function to collect only specific messages. Returns true for messages to be collected, false otherwise.
Type: function
options.time Optional: The duration in milliseconds for the collector to run (default: 60000 ms).
Type: number
options.max Optional: The maximum number of messages to collect (default: unlimited).
Type: number

Method:

handleMessage(message: object): void

Handles a new message received by the collector and adds it to the collected messages if it meets the filter criteria. Emits a "collected" event with the collected messages and count.

Example:


const { TelegramBot } = require("telegramsjs");
const bot = new TelegramBot("BOT_TOKEN");

bot.on("message", (message) => {
  const messageCollector = new MessageCollector({
    chatId: 123456789, // Replace with the ID of the chat you want to collect messages from
    filter: (message) => message.text && message.text.length > 0, // A filter function to collect only messages with text
    time: 60000, // Optional: the duration in milliseconds for the collector to run (default: 60000 ms)
    max: 10, // Optional: the maximum number of messages to collect (default: unlimited)
  });
  
  messageCollector.handleMessage(message);
});

bot.login();
