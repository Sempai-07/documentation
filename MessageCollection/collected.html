collected - Telegram Bot API

Description:

This method returns the collected messages from the MessageCollector. It retrieves an array of strings containing the messages that have been collected by the collector.

Return Value:

This method returns an array of strings representing the collected messages.

Example:


bot.on("message", (message) => {
  const messageCollector = new MessageCollector({
    chatId: 123456789, // Replace with the ID of the chat you want to collect messages from
    filter: (message) => message.text && message.text.length > 0, // A filter function to collect only messages with text
    time: 60000, // Optional: the duration in milliseconds for the collector to run (default: 60000 ms)
    max: 10, // Optional: the maximum number of messages to collect (default: unlimited)
  });
  
  messageCollector.handleMessage(message);
  
  console.log(messageCollector.collected());
});
