setMax - Telegram Bot API

Description:

This method sets a new maximum number of messages to collect for the MessageCollector. It allows you to update the maximum number of messages that the collector can collect.

Parameters:

Parameter Description
max The new maximum number of messages to set.
Type: number

Return Value:

This method returns a boolean value. It returns `true` if the maximum number is set successfully, and `false` otherwise.

Example:


bot.on("message", (message) => {
  const messageCollector = new MessageCollector({
    chatId: 123456789, // Replace with the ID of the chat you want to collect messages from
    filter: (message) => message.text && message.text.length > 0, // A filter function to collect only messages with text
    time: 60000, // Optional: the duration in milliseconds for the collector to run (default: 60000 ms)
    max: 6, // Optional: the maximum number of messages to collect (default: unlimited)
  });
  
  messageCollector.handleMessage(message);
  
  if (messageCollector.count() > 5) {
   messageCollector.setMax(10);
  }
});
