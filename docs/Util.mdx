# Telegram Bot Utility Functions

This document provides documentation for various utility functions related to working with Telegram bot messages.

## `checkMessageLinks`

Checks if a message contains any links.

### Parameters:

| Parameter | Description           |
| --------- | --------------------- |
| `message` | The message to check. |

### Return Value:

Returns `true` if the message contains links, otherwise `false`.

## `extractUserMentions`

Extracts user mentions from a message.

### Parameters:

| Parameter | Description                           |
| --------- | ------------------------------------- |
| `message` | The message to extract mentions from. |

### Return Value:

An array of user mentions found in the message.

## `extractHashtags`

Extracts hashtags from a message.

### Parameters:

| Parameter | Description                           |
| --------- | ------------------------------------- |
| `message` | The message to extract hashtags from. |

### Return Value:

An array of hashtags found in the message.

## `checkLocation`

Checks if a location object is valid.

### Parameters:

| Parameter            | Description                    |
| -------------------- | ------------------------------ |
| `location`           | The location object to check.  |
| `location.latitude`  | The latitude of the location.  |
| `location.longitude` | The longitude of the location. |

### Return Value:

Returns `true` if the location object is valid, otherwise `false`.

## `extractUserIdFromLink`

Extracts the user ID from a Telegram link.

### Parameters:

| Parameter | Description                                    |
| --------- | ---------------------------------------------- |
| `link`    | The Telegram link to extract the user ID from. |

### Return Value:

The extracted user ID or `null` if not found.
