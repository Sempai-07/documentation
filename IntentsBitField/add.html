IntentsBitField "add" Method

Description:

Use this method to add one or more bits to the IntentsBitField. The "add" method takes one or more numbers (bits) as arguments and updates the IntentsBitField instance by setting the corresponding bits to 1.

Parameters:

...ints: number[] - The bits to add to the IntentsBitField.

Returns:

The updated IntentsBitField instance after adding the specified bits.

Throws:

BitFieldError - If a specified bit is not a number.

Example:


// Example of using the "add" method to update the IntentsBitField

const { IntentsBitField, IntentBits, TelegramBot } = require("telegramsjs");

const intents = new IntentsBitField();

// Example of adding IntentBits.Message and IntentBits.EditedMessage to the IntentsBitField
try {
  intents.add(
    IntentBits.Message,
    IntentBits.EditedMessage
  );
  console.log(intents.serialize()); // Output: 3
} catch (error) {
    console.error(`Error: ${error.message}`);
}

const bot = new TelegramBot("BOT_TOKEN", {
  intents: intents.serialize()
});

bot.login();
