IntentsBitField "serialize" Method

Description:

Use this method to return the IntentsBitField as a number. The "serialize" method returns the underlying bitfield value as a single number.

Returns:

The bitfield value as a number.

Example:


// Example of using the "serialize" method to get the bitfield value as a number

const { IntentsBitField } = require("telegramsjs");

const intents = new IntentsBitField(7);
const bitfieldValue = intents.serialize();
console.log(bitfieldValue); // Output: 7
