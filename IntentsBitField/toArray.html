IntentsBitField "toArray" Method

Description:

Use this method to return an array of intent strings based on the current bit value in the IntentsBitField. The "toArray" method iterates through the IntentBits enum and checks if each intent is present in the IntentsBitField instance. If present, it adds the corresponding intent string to the resulting array.

Returns:

An array of intent strings based on the current bit value in the IntentsBitField.

Example:


// Example of using the "toArray" method to get an array of intent strings

const { IntentsBitField, IntentBits } = require("telegramsjs");

const intents = new IntentsBitField();

// Example of adding IntentBits.Message and IntentBits.EditedMessage to the IntentsBitField
intents.add(
  IntentBits.Message,
  IntentBits.EditedMessage
);

const intentArray = intents.toArray();
console.log(intentArray); // Output: [ 'message', 'edited_message' ]
