decodeIntents Function

Description:

Use this function to decode a bit field instance and return an array of intent strings. The "decodeIntents" function takes an instance of IntentsBitField as an argument and iterates through the IntentBits enum. It checks if each intent is present in the IntentsBitField instance and adds the corresponding intent string to the resulting array.

Parameters:

intentsBitField: IntentsBitField - The bit field instance to decode.

Returns:

An array of intent strings decoded from the IntentsBitField.

Example:


// Example of using the "decodeIntents" function to get an array of intent strings

const { decodeIntents, IntentBits, IntentsBitField } = require("telegramsjs");

const intents = new IntentsBitField().add(IntentBits.Message);
const decodedIntents = decodeIntents(intents);
console.log(decodedIntents); // Output: ['message']

const emptyIntents = new IntentsBitField();
const decodedEmptyIntents = decodeIntents(emptyIntents);
console.log(decodedEmptyIntents); // Output: []
