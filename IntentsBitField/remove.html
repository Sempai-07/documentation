IntentsBitField "remove" Method

Description:

Use this method to remove one or more Telegram intents from the IntentsBitField. The "remove" method takes one or more numbers (bits) as arguments and updates the IntentsBitField instance by clearing the corresponding bits to 0.

Parameters:

...ints: number[] - The bits that represent the Telegram intents to be removed from the IntentsBitField.

Returns:

The updated IntentsBitField instance after removing the specified intents.

Throws:

BitFieldError - If an invalid argument is passed.

Example:


// Example of using the "remove" method to update the IntentsBitField

const { IntentsBitField, TelegramBot } = require("telegramsjs");

const intents = new IntentsBitField(7);

// Example of removing intents 2 and 4 from the IntentsBitField
try {
  intents.remove(2, 4);
  console.log(intents.serialize()); // Output: 1
} catch (error) {
    console.error(`Error: ${error.message}`);
}

const bot = new TelegramBot("BOT_TOKEN", {
  intents: intents.toArray()
});

bot.login();
