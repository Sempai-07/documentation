Markup - Telegram Bot API

Description:

The `Markup` class is used to create and handle various reply markups for Telegram bot messages. It provides methods to set text, selective behavior, placeholder, resize keyboard, one-time keyboard, remove keyboard, force reply, and build custom keyboards and inline keyboards with various options.

Methods:

  • setText(text: string): this - Sets the text of the reply markup.

  • selective(value = true): this - Sets the selective behavior of the reply markup.

  • placeholder(placeholder: string): this - Sets the input field placeholder of the reply markup.

  • resize(value = true): this - Sets whether the reply markup should be resized.

  • oneTime(value = true): this - Sets whether the reply markup is a one-time keyboard.

  • removeKeyboard(): this - Sets the reply markup to remove the keyboard.

  • forceReply(): this - Sets the reply markup to force a reply.

  • keyboard(buttons: Button[], options?: WrapOptions): this - Sets the custom keyboard using an array of buttons.

  • inlineKeyboard(buttons: Button[], options?: WrapOptions): this - Sets the inline keyboard using an array of buttons.

  • buildKeyboard(buttons: Button[], options: BuildKeyboard): Button[][] - Builds the keyboard array based on the provided options.

  • static text(text: string): Text - Creates a `Text` object with the provided text. This static method returns a `Text` object that can be used as a part of the reply markup.

  • static contactRequest(text: string): KeyboardButton.RequestContactButton & Text - Creates a contact request button with the provided text. This static method returns a `KeyboardButton.RequestContactButton` object along with the text.

  • static locationRequest(text: string): KeyboardButton.RequestLocationButton & Text - Creates a location request button with the provided text. This static method returns a `KeyboardButton.RequestLocationButton` object along with the text.

  • static pollRequest(text: string, type?: "quiz" | "regular"): KeyboardButton.RequestPollButton & Text - Creates a poll request button with the provided text and an optional type ("quiz" or "regular"). This static method returns a `KeyboardButton.RequestPollButton` object along with the text and the poll request type.

  • static userRequest(text: string, request_id: number, user_is_bot?: boolean, user_is_premium?: boolean): KeyboardButton.RequestUserButton & Text - Creates a user request button with the provided text, request ID, and optional user details. This static method returns a `KeyboardButton.RequestUserButton` object along with the text and user request details.

  • static chatRequest(text: string, request_id: number, chat_is_channel: boolean, args: {...}): KeyboardButton.RequestChatButton & Text - Creates a chat request button with the provided text, request ID, chat type (channel or forum), and other optional chat details. This static method returns a `KeyboardButton.RequestChatButton` object along with the text and chat request details.

  • static url(text: string, url: string): InlineKeyboardButton.UrlButton & Text - Creates a URL button with the provided text and URL link. This static method returns an `InlineKeyboardButton.UrlButton` object along with the text and URL.

  • static callback(text: string, data: string): InlineKeyboardButton.CallbackButton & Text - Creates a callback button with the provided text and callback data. This static method returns an `InlineKeyboardButton.CallbackButton` object along with the text and callback data.

  • static switchToChat(text: string, value: string): InlineKeyboardButton.SwitchInlineButton & Text - Creates a switch to chat button with the provided text and value. This static method returns an `InlineKeyboardButton.SwitchInlineButton` object along with the text and switch value.

  • static switchToCurrentChat(text: string, value: string): InlineKeyboardButton.SwitchInlineCurrentChatButton & Text - Creates a switch to current chat button with the provided text and value. This static method returns an `InlineKeyboardButton.SwitchInlineCurrentChatButton` object along with the text and switch value.

  • static game(text: string): InlineKeyboardButton.GameButton & Text - Creates a game button with the provided text. This static method returns an `InlineKeyboardButton.GameButton` object along with the text.

  • static pay(text: string): InlineKeyboardButton.PayButton & Text - Creates a pay button with the provided text. This static method returns an `InlineKeyboardButton.PayButton` object along with the text.

  • static login(text: string, url: string, args?: {...}): InlineKeyboardButton.LoginButton & Text - Creates a login button with the provided text, URL link, and other optional login details. This static method returns an `InlineKeyboardButton.LoginButton` object along with the text and login details.

  • static webApp(text: string, url: string): InlineKeyboardButton.WebAppButton & Text - Creates a web app button with the provided text and URL link. This static method returns an `InlineKeyboardButton.WebAppButton` object along with the text and URL. 


const { Markup, TelegramBot } = require("telegramsjs");

const bot = new TelegramBot("TOKEN_BOT");

const markup = {
  inline_keyboard: [[Markup.callback("Marmok", "data"), Markup.url("Last video", "https://youtu.be/J6a57KEV6hY")]]
}

bot.command("button", (ctx) => {
  ctx.reply("Markup test", {
    reply_markup: markup
  });
});

bot.action("data", (ctx) => {
  ctx.answerCallbackQuery({
    text: 'Thank you!'
  }).catch(err => console.log);
});

bot.login();
